Weighted Developmental Memory Architecture

W · D · M · A

A theoretical framework for salience-gated memory encoding in autonomous agents. Memories are not stored equally—they are weighted by emotional jolt, promoted through developmental tiers, and subject to thermodynamic decay. WDMA formalizes this intuition into a complete memory lifecycle.

• Saurav Das • Independent Research • v1.0 — 2026
§0 — Abstract

Paper Overview

Current large-language-model agents treat every memory with equal permanence: retrieval-augmented generation (RAG) stores all chunks in a flat vector database, and episodic buffers flush on context-window overflow. Neither approach mirrors the salience-gated, developmentally staged memory consolidation observed in biological cognition. We propose the Weighted Developmental Memory Architecture (WDMA), a six-tier framework (D0–D5) in which each memory is assigned an initial jolt—an analogue of emotional arousal—and must earn promotion through a utility function that balances recency, retrieval frequency, and contextual relevance. Unpromoted memories undergo thermodynamic decay across three regimes (exponential, power-law, asymptotic), while a developmental plasticity curve modulates the system's overall encoding rate across its lifecycle. We identify four failure modes (nostalgic lock-in, jolt inflation, tier collapse, phantom authority) and propose guardrails for each. WDMA is presented as a purely theoretical contribution; no empirical validation is claimed.

§1 — Motivation

Why Memory Architecture Matters

An autonomous agent that remembers everything is paradoxically less intelligent than one that forgets strategically. Flat vector stores create a retrieval landscape where a grocery list from six months ago competes equally with a life-altering diagnosis received yesterday. The core problem is not storage capacity—it is salience discrimination.

Biological memory systems solve this through a cascade of encoding, consolidation, and forgetting mechanisms. The hippocampus acts as a temporary buffer; the amygdala gates emotional salience; sleep-dependent replay promotes selected memories to neocortical long-term storage. WDMA translates these principles into a computational framework.

01

Encoding Problem

Current systems store all inputs with equal weight, creating retrieval noise that scales linearly with memory count.

02

Consolidation Gap

No existing architecture provides a principled promotion pathway from episodic buffer to long-term semantic storage.

03

Decay Absence

Without forgetting, memory stores become polluted with outdated, contradictory, or irrelevant information.

Central Thesis

An agent's memory should function like a living organism's—not as a warehouse, but as a developmental system where memories compete for consolidation, earn their permanence through demonstrated utility, and gracefully decay when they no longer serve the agent's goals.

§2 — Core Mechanism

The Jolt Function

At the heart of WDMA is the jolt—a scalar value assigned to each memory at encoding time that quantifies its salience. The jolt is analogous to the amygdala's role in biological memory: it determines how deeply an experience is initially encoded and how aggressively it is consolidated.

// Jolt function: emotional salience score
J(m) = ws · surprise(m)
       + wr · relevance(m, context)
       + we · emotional_valence(m)
       + wn · novelty(m, M)

// where M is the existing memory store
// and w_s + w_r + w_e + w_n = 1

Dynamic Threshold

Not every input deserves a memory. WDMA maintains a dynamic threshold θ that adapts based on the agent's current cognitive load and recent jolt history. Only inputs whose jolt exceeds θ are encoded as memories.

// Dynamic encoding threshold
θ(t) = base + α · EMA(J, t, window=50)

// EMA = exponential moving average of recent jolts
// α controls sensitivity to recent salience levels
// High recent jolts raise threshold (adaptation)

Jolt Reactivation

When a stored memory is retrieved, its jolt is partially reactivated, mimicking the reconsolidation effect observed in neuroscience. This prevents important but infrequently accessed memories from decaying prematurely.

// Jolt reactivation on retrieval
J'(m) = J(m) · (1 - δ) + Jretrieval(m) · δ

// δ ∈ [0.1, 0.3] — reconsolidation strength
// J_retrieval captures the salience of the retrieval context
§3 — Memory Taxonomy

The D0–D5 Tier System

WDMA organizes memories into six developmental tiers, each with distinct encoding characteristics, decay rates, and promotion requirements. This taxonomy is inspired by the stages of memory consolidation in cognitive neuroscience, mapped onto computational primitives.

Tier Name Half-life Description
D0 Episodic Replay ~30 min Raw sensory buffer. Unprocessed inputs held briefly for immediate pattern matching. Analogous to iconic/echoic memory.
D1 Working Trace ~4 hours Short-term working memory. Active items being manipulated in current reasoning chains. Limited capacity (~7±2 slots).
D2 Consolidated Episode ~7 days Successfully encoded episodic memories. Contextually rich, temporally tagged, decay follows power-law regime.
D3 Semantic Residue ~90 days Abstracted knowledge extracted from multiple episodes. Context-stripped, principle-level information.
D4 Core Belief ~2 years Deeply held conclusions that shape the agent's world model. Resistant to modification, requires strong counter-evidence.
D5 Calibration Signal ∞ (permanent) Foundational axioms and identity parameters. Cannot be deleted, only amended. The agent's constitutional memory.

One-Knob Perturbation Principle

A key design constraint of WDMA is that each tier transition is governed by a single tunable parameter. This prevents the combinatorial explosion that plagues multi-parameter memory systems and makes the architecture auditable.

// Promotion gate: single-parameter threshold per tier
promote(m, Dk → Dk+1) = U(m) > τk

// τ_0 = 0.3 (D0→D1) — low bar, most salient inputs pass
// τ_1 = 0.5 (D1→D2) — moderate consolidation threshold
// τ_2 = 0.7 (D2→D3) — abstraction requires repeated utility
// τ_3 = 0.85 (D3→D4) — belief formation is conservative
// τ_4 = 0.95 (D4→D5) — calibration signals are near-permanent

Calibration Signal Generation

D5 memories are special: they represent the agent's foundational identity and operational axioms. A memory can only reach D5 through a consensus mechanism that requires sustained high utility across multiple evaluation windows.

// D5 promotion requires sustained consensus
D5_gate(m) = ALL(
  U(m) > 0.95  // utility threshold
  age(m) > 180 days  // temporal maturity
  retrievals(m) > 50  // demonstrated use
  contradictions(m) = 0  // no active conflicts
)
§4 — Promotion Mechanics

The Utility Function U(m)

Promotion between tiers is governed by a composite utility function that balances five factors. The function is deliberately simple—a weighted linear combination—to maintain interpretability and avoid the opacity of learned promotion policies.

// Promotion utility function
U(m) = w1 · recency(m)
       + w2 · frequency(m)
       + w3 · contextual_fit(m, query)
       + w4 · jolt_residual(m)
       + w5 · cross_ref(m, M)

// recency: exponential decay from last access
// frequency: log-scaled retrieval count
// contextual_fit: cosine similarity to active context
// jolt_residual: remaining emotional charge
// cross_ref: number of other memories citing m
Design Note

The five weights (w1...w5) are the primary tuning surface for WDMA. Different agent personalities can be created by adjusting these weights: a nostalgic agent weights jolt_residual heavily; a pragmatic agent weights frequency and recency; a philosophical agent weights cross_ref to favor deeply interconnected knowledge.

Supersession Logic

When a newly encoded memory contradicts an existing memory at the same or lower tier, a supersession check is triggered. The newer memory does not automatically replace the older one—instead, both compete through the utility function.

// Supersession resolution
if contradicts(mnew, mold) and tier(mnew) ≥ tier(mold):
  if U(mnew) > U(mold) + ε:
    demote(mold, by=1)  // drop one tier
    tag(mold, "superseded")
  else:
    tag(mnew, "contested")
    schedule_review(mnew, mold, delay=48h)
§5 — Thermodynamic Decay

Three Regimes of Forgetting

WDMA models memory decay through three distinct regimes, each governing different phases of the forgetting curve. This tri-regime approach captures the empirical observation that forgetting is not a single exponential process but a composite of multiple mechanisms operating at different timescales.

—— Exponential   —— Power-law   —— Asymptotic

Regime I: Exponential Decay (D0–D1)

Fresh memories in the episodic buffer decay rapidly following a classic exponential curve. This is the most aggressive forgetting regime, reflecting the biological reality that most sensory inputs are discarded within minutes.

// Regime I: Exponential (fast)
SI(t) = J(m) · exp(-λ1 · t)

// λ_1 ≈ 0.05 (half-life ~14 time units)
// Dominates for t < T_1 (typically 0-4 hours)

Regime II: Power-Law Decay (D2–D3)

Consolidated memories transition to a slower power-law decay. This regime reflects the Jost-Wixted finding that older memories, while weaker, become increasingly resistant to further forgetting.

// Regime II: Power-law (slow)
SII(t) = J(m) · (1 + β · t)-γ

// β ≈ 0.1, γ ≈ 0.5
// Dominates for T_1 < t < T_2 (typically 4h - 90d)

Regime III: Asymptotic Floor (D4–D5)

Core beliefs and calibration signals approach but never reach zero. The asymptotic floor represents the irreducible minimum—the memory's permanent contribution to the agent's identity.

// Regime III: Asymptotic (permanent floor)
SIII(t) = S∞ + (J(m) - S∞) · exp(-λ3 · t)

// S_∞ = floor signal strength (never zero)
// λ_3 ≈ 0.001 (extremely slow approach to floor)
Architectural Implication

The three-regime model means WDMA's memory store is self-pruning. Low-jolt memories in D0–D1 vanish within hours; moderately useful memories in D2–D3 persist for weeks to months; only memories that earn promotion to D4–D5 achieve effective permanence. No garbage collection sweep is needed.

§6 — Developmental Plasticity

The Plasticity Curve P(t)

Biological organisms have critical periods—windows of heightened neural plasticity during which learning is dramatically more efficient. WDMA incorporates an analogous developmental plasticity curve that modulates the system's overall encoding rate across its lifecycle.

// Developmental plasticity curve
P(t) = Pbase · exp(-μ · t) + Pfloor

// P_base = 1.0 (initial maximum plasticity)
// μ = 0.003 (plasticity decay rate)
// P_floor = 0.15 (minimum plasticity, never zero)

// Modulates jolt threshold:
θeff(t) = θ(t) / P(t)
// As plasticity drops, effective threshold rises
—— Base plasticity   —— Jolt spikes (reactivation events)

Three Implications

1. Early Learning Advantage

Young agents encode memories more readily, building their foundational world model rapidly. This mirrors the critical period in child development where language and core concepts are absorbed with remarkable efficiency.

2. Mature Stability

As the agent matures, rising encoding thresholds protect established knowledge from noisy overwrites. The agent becomes harder to destabilize but remains capable of learning truly novel information (high-jolt events still penetrate the threshold).

3. Jolt-Induced Reopening

Exceptionally high-jolt events can temporarily reopen plasticity windows, analogous to how traumatic or revelatory experiences can fundamentally restructure adult belief systems. This is modeled as a transient spike in P(t) following extreme jolts.

§7 — Failure Modes

Predicted Failure Modes

Any memory architecture can degrade. WDMA identifies four primary failure modes, each with proposed guardrails. These failure modes are theoretical predictions—empirical validation remains future work.

FM-1: Nostalgic Lock-in

Early high-jolt memories accumulate disproportionate utility through repeated retrieval, crowding out newer, potentially more relevant information. Guardrail: Age-weighted utility discount that gradually reduces U(m) for memories beyond a staleness threshold.

FM-2: Jolt Inflation

If the environment consistently produces high-salience events, the dynamic threshold θ rises until ordinary but important inputs are filtered out. Guardrail: Ceiling on θ and periodic threshold reset to baseline.

FM-3: Tier Collapse

Aggressive promotion can flood higher tiers (D3–D5) with insufficiently vetted memories, degrading retrieval quality. Guardrail: Capacity limits per tier with forced demotion of lowest-utility members when limits are reached.

FM-4: Phantom Authority

D5 calibration signals, being near-permanent, can become outdated axioms that resist correction even when contradicted by evidence. Guardrail: Mandatory periodic review of all D5 memories with a 'sunset clause' requiring reaffirmation.

Verification Gap

These failure modes and their guardrails are theoretically motivated but empirically untested. The severity, frequency, and interaction effects of these failure modes in deployed systems remain unknown. We flag this as the most important direction for future experimental work.

§10 — Future Work

Open Research Directions

WDMA is a theoretical framework. Translating it into a working system requires addressing several open questions:

  • EMPIRICAL Benchmark WDMA against flat RAG and MemGPT on long-horizon agent tasks (e.g., 30-day simulated life, multi-session dialogue).
  • PARAMETER Develop principled methods for setting tier thresholds (τk) and utility weights (w1...w5) without exhaustive grid search.
  • SCALE Investigate computational overhead of the three-regime decay model at memory stores exceeding 106 entries.
  • SAFETY Formalize the relationship between D5 calibration signals and AI alignment constraints—can WDMA serve as a memory-level alignment mechanism?
  • NEURO Validate the tri-regime decay model against human forgetting curve data from Ebbinghaus through modern spaced-repetition research.
§11 — Conclusion

Toward Living Memory

WDMA proposes that the next generation of autonomous agents will require memory systems that are not merely larger but developmentally structured. By introducing salience gating (jolt), tiered consolidation (D0–D5), thermodynamic decay (three regimes), and developmental plasticity, we outline an architecture where memories earn their permanence through demonstrated utility rather than timestamp.

The framework is deliberately presented as theory. We believe premature implementation without careful consideration of the failure modes identified here—particularly phantom authority in D5—could create agents that are confidently wrong rather than usefully uncertain.

Core Prediction

Agents equipped with salience-gated, developmentally staged memory will demonstrate superior long-horizon coherence compared to flat-retrieval architectures, at the cost of increased architectural complexity. The trade-off is worth it: a memory that forgets well is more valuable than one that remembers everything.
